# AWS-Serverless-Project
Serverless resume collection system built with AWS. Frontend hosted on GitHub Pages and backend powered by S3, Lambda, API Gateway, and DynamoDB to automatically process and store candidate resumes.

AWS Serverless Resume Collection System
📌 Project Overview

This project demonstrates a Serverless Resume Collection System built using AWS cloud services.

The system allows candidates to upload resumes through a simple HTML frontend, while AWS services automatically process and store the data.

The goal of this project is to show how startup companies can collect and manage job applications with minimal infrastructure and near-zero cost using the AWS Free Tier.

🚀 Architecture

Workflow of the system:

Candidate
   ↓
HTML Upload Page (GitHub Pages)
   ↓
API Gateway
   ↓
AWS Lambda
   ↓
Amazon S3 (Resume Storage)
   ↓
Amazon DynamoDB (Candidate Data Storage)


🛠 Technologies Used
Frontend

HTML

GitHub Pages

Backend

Amazon S3 – Stores uploaded resumes

AWS Lambda – Processes resume uploads

Amazon DynamoDB – Stores candidate details

Amazon API Gateway – Provides API endpoint

Programming

Python (Boto3)
